Keyboard shortcuts
------------------
right arrow : rotate right around y axis
left arrow : rotate left around y axis
up arrow : rotate up around x axis
down arrow : rotate down around x axis
f : zoom in
z : increase s parameter
x : decrease s parameter
a : increase b parameter
s : decrease b parameter
q : increase r parameter
w : decrease r parameter

How to run
----------
make
./hw1
